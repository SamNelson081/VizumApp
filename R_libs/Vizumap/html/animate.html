animate {Vizumap}R Documentation

Animate a pixel map

Description

This function animates a pixel map created with build_pmap so that the pixels ficker between a series of sampled values.

Usage

animate(pmap, flickerSpeed = 0.05, aniLength)

Arguments

pmap

An object from build_pmap.

flickerSpeed

A numeric value between 0 and 0.5.

aniLength

An integer. A larger value corresponds with a longer animation; a value of at least 20 is recommended.
